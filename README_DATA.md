# Dataset disclaimer

`transactions.csv` is **100% synthetic**. It was generated by `Claude files/generate_transactions.py` to reproduce the statistical behaviour of a global SaaS payments book (~$2B ARR, 14 PSPs, 30 countries) for demo and pattern-analysis purposes.

- No row corresponds to a real transaction, customer, merchant, card, bank account, or payment.
- Brand and product names (PSPs, processors, schemes, issuers, plan SKUs, the parent merchant "Cirrus Cloud Platform") are descriptive labels for a fictional book. They are not affiliated with, endorsed by, or sourced from any real company.
- The dataset was built from publicly available payments-industry documentation (PSP/orchestrator developer docs, scheme rule summaries, public benchmarks). It contains no proprietary, confidential, or non-public information from any employer or third party.
- Any resemblance to real merchants, incidents, outage windows, or contractual terms is coincidental.

Use of this dataset is for demonstration and educational purposes only.
